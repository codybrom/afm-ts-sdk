# Generation Options

Demonstrates configuring generation parameters like temperature, sampling mode, and token limits.

## What it shows

- Setting `temperature` for controlling randomness
- Using `SamplingMode.random()` with `top` (top-k) and `seed`
- Setting `maximumResponseTokens` to limit output length
- Passing options to `session.respond()`

## Run

```bash
npx tsx examples/generation-options/generation-options.ts
```

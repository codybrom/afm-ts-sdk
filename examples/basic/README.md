# Basic Text Generation

Demonstrates the simplest usage: create a model, check availability, open a session, and generate a text response.

## What it shows

- Creating a `SystemLanguageModel`
- Checking availability with `waitUntilAvailable()`
- Creating a `LanguageModelSession` with instructions
- Generating a response with `session.respond()`
- Proper cleanup with `dispose()`

## Run

```bash
npx tsx examples/basic/basic.ts
```

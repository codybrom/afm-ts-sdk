# JSON Schema Generation

Demonstrates generation using Apple's JSON Schema format via `respondWithJsonSchema()`.

## What it shows

- Building a schema with `GenerationSchema` and converting to JSON with `toDict()`
- Using `session.respondWithJsonSchema()` for JSON Schema-based generation
- Accessing the full generated object with `content.toObject()`

## Run

```bash
npx tsx examples/json-schema/json-schema.ts
```
